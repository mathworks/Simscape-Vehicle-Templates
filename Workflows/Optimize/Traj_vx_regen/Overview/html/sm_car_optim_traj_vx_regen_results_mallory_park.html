Minimize Lap Time Considering Regeneration Using Optimization, Mallory Park (No Elevation Change)

This example shows how to minimize lap times using optimization algorithms using the Mallory Park Racing Circuit. This optimization assumes no elevation to shorten the optimization process. The cost function considers the loss in battery state of charge to factor in vehicle range to the optimization.
